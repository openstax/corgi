.. _glossary:

========
Glossary
========

.. glossary::

   Concourse CI
      Known as "an open-source continuous thing-doer" it's a system for creating
      Continuous Integration (CI) pipelines.

   Collection
      The term typically used to refer to a book at OpenStax. A collection is a
      grouping of modules.

   COPS
      The term is used for a new web application, called Content Output Producer Service, to produce
      various outputs of our OpenStax book content.
